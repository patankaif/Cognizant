Bootstrap Directory & Sass Compilation

Directory Structure
  • css/: Contains style libraries (e.g. bootstrap.min.css) that layout websites.
  • js/: Contains Javascript scripts (e.g. bootstrap.bundle.min.js) that activate dropdowns, tabs, and modals.
  • icons/: Optional folder of vector shapes used to represent social and administrative logos.
Sass Customization Concepts

By installing bootstrap through npm, you can change variable defaults (like changing the primary blue color $primary to indigo or altering the default border curves $border-radius) inside a Sass file and recompile it into a new custom CSS stylesheet.
